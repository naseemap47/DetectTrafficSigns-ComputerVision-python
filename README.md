## Description
Detect Traffic Signs using TensorFlow and OpenCV.

## Requirements
#### Programming Language
* Python3 (pyCharm)
#### Libraries
* OpenCV
* TensorFlow
* Keras
* Numpy
* Matplotlib
* Pandas
* OS

## Project Setup
* Clone the project
* RUN - **detect_trafficSigns.py**
* Web-cam will become **ON** Automatically
* Detect Traffic Signs by Web-cam
  * Show Traffic Signs in front of Web-cam (Plane Surface)
  * It will tell which traffic sign is.
* To **Quit** - Press **Q-key**
